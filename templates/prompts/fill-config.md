# Fill-my-config prompt

Paste this to have the agent interview you and fill in `AGENTS.md`. It asks one question at a time — no editor needed.

```
Open the AGENTS.md file in this project folder. You are going to help me fill it in.

First, ask me whether this is a PERSONAL agent (my own productivity / a personal
project) or a COMPANY agent (a business process or workflow at my organisation).

Then interview me ONE question at a time to collect:
  - Agent name, my full name (and organisation, if Company)
  - My one clear goal, with a measurable target
  - My current state, described with at least one number
  - My deadline, if any
  - My reviewer's name and role, if any
  - 2–3 actions the agent IS allowed to do for this goal
  - 1–2 actions the agent must NEVER do
  - Any action that should require my explicit approval

After you have my answers, write them into the matching section of AGENTS.md,
replacing the placeholders, and delete the configuration block I'm not using.
Show me the filled-in result and ask me to confirm before saving. Do not invent
details — if something is unclear, ask me.
```
